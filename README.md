# Sistema de Gerenciamento de Academia

Sistema desenvolvido em **Java** com o objetivo de aplicar os principais conceitos da **Programação Orientada a Objetos (POO)** e reforçar boas práticas de desenvolvimento de software.

## 📋 Descrição

O sistema permite gerenciar o cadastro de alunos, instrutores e recepcionistas de uma academia, oferecendo funcionalidades para matrícula, gerenciamento de cadastros e geração de relatórios.

Este projeto foi desenvolvido como forma de praticar conceitos fundamentais da linguagem Java, como herança, polimorfismo, abstração e encapsulamento.

## 🚀 Funcionalidades

- Cadastro de alunos, instrutores e recepcionistas;
- Alteração e remoção de cadastros;
- Matrícula de alunos com ou sem um instrutor específico;
- Associação automática do aluno ao instrutor escolhido;
- Geração automática do número de matrícula de cada aluno;
- Consulta de relatórios contendo:
  - Total de alunos cadastrados;
  - Total de instrutores;
  - Total de recepcionistas;
  - Quantidade de alunos vinculados a cada instrutor;
- Menu interativo para navegação pelo sistema.

## 🛠️ Tecnologias utilizadas

- Java
- Programação Orientada a Objetos (POO)

## 📚 Conceitos aplicados

Durante o desenvolvimento foram utilizados diversos recursos da linguagem Java, entre eles:

- Abstração;
- Encapsulamento;
- Herança;
- Polimorfismo;
- Interfaces;
- Enumerações (`enum`);
- Métodos e atributos estáticos;
- Organização em classes e pacotes.

## 📌 Enum utilizado

O projeto utiliza um `enum` para representar o status da matrícula do aluno:

- Ativo
- Inativo
- Suspenso

## 💰 Interface para cálculo de salário

Foi utilizada uma interface para definir o cálculo do salário dos:

- Instrutores;
- Recepcionistas.

## 🎯 Objetivo do projeto

O principal objetivo deste projeto foi consolidar conhecimentos em Java e Programação Orientada a Objetos por meio da implementação de um sistema completo, aplicando conceitos de reutilização de código, organização, modularização e manutenção.
